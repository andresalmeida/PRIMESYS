# PRIMESYS — Claude Code Context

## Project
Next.js 14 website for PRIMESYS, an Ecuadorian technology consultancy with 25+ years of experience. Live at primesys.ec. Deployed to GitHub Pages via GitHub Actions.

## Stack
- Next.js 14 (static export to `dist/`)
- Tailwind CSS + CSS Modules (App.css, Styles/servicios.css)
- Framer Motion + GSAP for animations
- EmailJS for contact form
- Lucide React icons

## Design Context

### Users
Mixed B2B audience: enterprise CTOs/IT directors, SMB owners/startup founders, and public sector buyers in Ecuador and Latin America. Must feel credible to serious enterprise buyers while remaining clear to smaller clients.

### Brand Personality
**Trustworthy · Bold · Innovative** — premium, engineering-led credibility. The confidence of Nubank, the seriousness of Visa, the clarity of C6Bank. 25 years of real enterprise delivery behind every pixel.

### References
Positive: Nubank, C6Bank, Creditas, Visa, Discover USA.
Anti-references: Generic IT agency sites, dark-mode tech sites with teal/cyan neon accents and glassmorphism.

### Aesthetic Direction
- Dark navy (#212954) + Red (#FF4F46) core palette. Red is the ONLY accent.
- Remove all cyan, teal, and competing gradient bleeds from the design.
- Space Grotesk (headings) + Sora (body) + IBM Plex Mono (code/terminal only).
- Editorial, precise, measured confidence. Enterprise-grade, not startup-cool.

### Design Principles
1. **Red earns attention** — every other color defers to #FF4F46. No cyan, teal, gradient text, or competing glows.
2. **Hierarchy over decoration** — floating orbs, glassmorphism, and blur blobs that don't communicate information should be replaced with structure.
3. **Enterprise credibility at first glance** — restraint and precision build trust faster than motion and effects.
4. **Consistency across pages** — one design language, one font stack (no Arial in servicios.css), everywhere.
5. **Typography does the heavy lifting** — Space Grotesk at large scale is authoritative; let type set the tone.
